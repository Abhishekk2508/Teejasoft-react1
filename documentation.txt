==================== 📌 1. Project Overview ========================

The TeejaSoft Business Website is a modern, fully responsive corporate website developed using React JS.
It is designed to represent the company’s digital presence, showcase services, portfolio, client testimonials, blogs, and provide essential business information in an interactive and user-friendly manner.

The project follows a component-based architecture, ensuring scalability, maintainability, and performance optimization.
This website represents Teejasoft’s:

-> Company Profile/ Home page / Landing Page
-> Services (4 pgs) --- (Digital Marketing, Graphic Design, Paid Ads, Web Development)
-> About us 
-> Portfolio 
-> Blogs
-> Contact Us

==================== 🚀 6. Installation & Setup (Local Environment) ========================

Step 1: Clone Repository from
-> git clone https://github.com/Abhishekk2508/Teejasoft-react1.git

Step 2: Navigate to Project
-> cd Teejasoft-react1

Step 3: Install Dependencies
-> npm install

Step 4: Run Project
-> npm run dev

Project runs on: http://localhost:5173
